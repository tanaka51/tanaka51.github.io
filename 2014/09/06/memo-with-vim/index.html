Vim で memo をとる環境を(漸く)整えた

Sep 6, 2014   #vim 

ブログのデザインを新しくしたら更新するんだ、と思って早1年と半年。 漸く重い腰をあげて Octopress 用のテーマを探しにいったら、良い感じにシンプルなやつ があったので、これだ！とブログをいじりはじめたのでした。

そして久しぶりの更新。

最後の記事が Emacs の記事、今では Vim を使う毎日なのであった。
半年前から使い始めた Vim で memo をとる環境を(漸く)整えた整えたのでメモ。

memolist.vim + ctrlp

基本的にはこれを参考に設定しただけ。

Big Sky :: 2012年のVimメモ取り環境はmemolistとctrlpに決まりだ

合わせて、memo を Dropbox 経由で同期したいので以下の設定を追加。

let g:memolist_path = "~/Dropbox/memo"

memo 開くのとか、memo から grep するのとか、memo 一覧から fuzzy 検索するのとか、便利。

久しぶりの blog なので、こんなメモ程度で終わり。もっと駄文を増やしていきたい気持ち。
