uninstall_rvm

Jan 7, 2013  

rvm のアンインストール方法にちょっとはまったのでメモ。

やりたいこと

Ruby 環境の構築時に、誤って sudo 付きで rvm を入れてしまったので、消したい。

方法

$ sudo rvm implode

したら、何故かrvmコマンドが見つからない！って起こられたので、以下の方法で消した。

$ sudo rm -rf /usr/local/rvm
$ suod rm /etc/profile.d/rvm.sh

このあと、 ログアウト→ログイン をすること。

その後、

$ \curl -L https://get.rvm.io | bash -s stable --ruby

すればおｋ。

sudo で入れた rvm が残っていると、sudo 無し rvm を入れようとした時に、/usr/local 以下にインストールしにいこうとして失敗する。
