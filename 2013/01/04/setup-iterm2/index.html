Setup iTerm2

Jan 4, 2013  

iTerm2 を Emacs 仕様にする

Command を Meta にする。ただし、Command + Tab や Command * Space は有効にする。

  • iTerm -> Preferences… で設定画面を開く
  • Profiles -> Keys の下段の Left Option を +ESC にチェックする
  • Keys タブの Remap Modifier Keys の Left command key と Right command key をLeft Option に変える
  • Keys タブの Global Shortcut Keys の + をクリック、Action から Do Not Remap Modifires を選択、Command + Tab と Command + Space を登録する(必要があれば他のキーも登録する)

これで、tmux の Emacs モードも快適に使える:)
