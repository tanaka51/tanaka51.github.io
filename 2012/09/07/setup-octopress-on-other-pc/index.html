他の PC で Octopress をセットアップする

Sep 7, 2012  

他の PC で octopress の記事をポストしたいときの手順

前提条件

  • Github Pages を使っている
  • どこかで setup が終了している

やり方

  • git@github.com:username/username.github.com からクローン
  • source ブランチをチェックアウト $ git checkout -b source origin/source
  • master ブランチはいらないので消す $ git branch -D master
  • bundle インストール
    • RVM やらの設定もやっておく $ bundle install
  • 重要 rake setup_github_pages を必ず実行する
    • これは新しくクローンしたリポジトリでは毎回実行すること
    • この作業によって、_deploy 以下にデプロイ用のリポジトリが作成される
    • やらないとデプロイできないので注意 $ rake setup_github_pages
  • 後はいつもと同じ手順でポスト可能
