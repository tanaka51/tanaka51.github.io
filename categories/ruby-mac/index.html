install ruby on mac

Feb 3, 2013  

Ruby 1.9.3-p374 を Mac にインストールしようとしたら一発でできなかったのでメモ。

  • OS: OSX 10.8.2 (moutain lion)
  • rvm: 1.18.5

3行でまとめると、

  • MacPorts をアンインストール
  • Homebrew に完全移行
  • rm -rf ~/.rvm/usr (不要な rvm pkg をアンインストール)

という事になる。

最新の Ruby のインストールは、Homebrew のライブラリが相性が良いのかなーという印象。正しいかどうかは分からない。

以後、長いけれども、作業メモ。
