hugoに移行した

May 4, 2016  

ちょっと時間ができたのでしばらく手を付けてなかったこのブログのジェネレータを hugo に移行した。世の中 golang プロダクトで溢れてきた感がある。 デザインは tanksuzuki/angels を元にちょっといじったものを使っている。(tanaka51/angels)

Octopress から hugo への移行は特に難しい事はなくて、テーマの選定・ CSS の調整に時間がかかった。ちょっと困った事として、いくつか記事の URL が変わってしまったんだけど、 hugo のドキュメント見ながら修正するが面倒なので、このままにしておく。

追記: よく見たら特に面倒でもなかったので、昔の URL と同じになるように設定した。具体的には、 config.yml の permalinks/post を /:year/:month/:day/:filename にして、 content/post 以下のファイル名から日付をとった。昨日までは /:year/:month?:day/:title にしていた。

はてさて、今年はどのくらいブログを更新するんだろうか。
